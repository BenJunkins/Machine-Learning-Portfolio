# Key Findings:

Preliminary term grades (G1 and G2) are highly correlated with final grades and are the strongest predictors of student success.

The best-performing model was a Linear Regression model that included G1 and G2 scores, achieving a Root Mean Squared Error (RMSE) of 1.88.

While models excluding preliminary grades had lower accuracy, they were recommended for use as early-warning indicators for low-resource Tier 1 interventions (e.g., advisor check-ins), while the more robust model was recommended for higher-tier interventions.
